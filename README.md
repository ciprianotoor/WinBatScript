# WinBatScript

## Descripción

`WinBatScript` es un repositorio de **scripts en Windows (.bat / .cmd)** creados para facilitar tareas de automatización personal y administrativa en entornos de usuario.
Los scripts incluyen operaciones de respaldo, menús rápidos, gestión de archivos y otros procedimientos repetitivos que pueden mejorar la eficiencia en entornos Windows.

> ⚠️ **Nota:** Estos scripts se proporcionan “tal cual” para fines educativos o de automatización personal. No se asume ninguna responsabilidad por daños o pérdida de datos derivados de su uso. Úsalos bajo tu propio riesgo.

---

## Contenidos del repositorio

Actualmente, el repositorio incluye:

* **`Backup_OneDrive_SD.bat`**
  Script para realizar copias de seguridad de carpetas específicas en OneDrive o unidades externas.

* **`Power_Menu.cmd`**
  Script que despliega un menú de opciones rápidas para tareas frecuentes en Windows.

> Puedes agregar tus propios scripts `.bat` siguiendo la misma estructura y convenciones.

---

## Instalación

1. Clona el repositorio en tu máquina:

```powershell
git clone git@github.com:ciprianotoor/WinBatScript.git
```

2. Navega al directorio del repositorio:

```powershell
cd WinBatScript
```

3. Ejecuta los scripts desde PowerShell o haciendo doble clic en los archivos `.bat` / `.cmd`.

---

## Uso

* Los scripts se ejecutan directamente en Windows, sin necesidad de software adicional.
